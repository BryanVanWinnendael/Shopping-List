# Shopping List - App

## Requirements

### Apple

Generate a p8 file.

### Firebase

Generate the google-services.json and GoogleService-Info.plist files for your project.

## Commands

```bash
eas build -p ios --auto-submit
```

```bash
eas build -p ios --profile production --auto-submit
```

```bash
npx expo start --no-dev --minify
```
